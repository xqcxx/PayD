# #009: Design Soroban Smart Contract for Vesting / Escrow Logic

**Category:** [CONTRACT]
**Difficulty:** ● HARD
**Tags:** `soroban`, `escrow`, `vesting`, `rust`

## Description

Draft a Soroban contract that holds salary in escrow and releases it on a schedule. Implement time-locked payments, cliff vesting, and early-exit clawback logic using Soroban's storage and auth framework.

## Acceptance Criteria

- [x] Soroban contract draft implemented in Rust.
- [x] Time-locked payment logic functional.
- [x] Cliff vesting and clawback mechanisms implemented.
- [x] Leverages Soroban's native auth and storage systems.
